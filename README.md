# UFF + Território

Site independente para apresentar a proposta pública **Programa UFF + Território: Observatório Popular do Médio Paraíba**.

## Stack

- Next.js
- TypeScript
- Tailwind CSS v4

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Deploy

O projeto está pronto para Vercel. Publique pelo painel da Vercel ou conecte o repositório GitHub.
